# 查询充值对象

通过结算对象的`id`查询一个已创建的结算对象。

#### 请求方法

GET `$endpoint/query/recharge/{id}`

#### 请求参数

| 字段名    | 类型   | 描述                                        | 属性     |
| --------- | ------ | ------------------------------------------- | -------- |
| id        | string | 充值对象ID，该参数仅需要填写在请求地址内。  | required |
#### 返回参数

返回一个支付对象。
