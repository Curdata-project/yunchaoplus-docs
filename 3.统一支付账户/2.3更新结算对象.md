# 更新结算对象

你可以通过该接口更新一个 `settle` 对象。

#### 请求方法

PUT `$endpoint/settle/{settls_id}`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| settls_id | string | 结算对象id | required |
| verify_code           | string      | 验证码                 | required      |
| ori_req_sn           | string      | 在创建C端结算对象时返回的流水号               | required      |


#### 返回参数

返回一个结算对象。
