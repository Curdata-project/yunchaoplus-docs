# 更新结算对象

你可以通过该接口更新一个 `settle` 对象。

#### 请求方法

PUT `$endpoint/settle/{settls_id}`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| settls_id | string | asdasdasd | required |
| extra           | object      | 额外参数，具体渠道不同有所区别，参见额外参数                 | optional      |


#### 返回参数

返回一个结算对象。
