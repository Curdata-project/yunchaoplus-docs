# 查询充值对象列表

返回之前创建过充值对象的一个列表。具体的查询参数遵循前述中描述的分页参数。

#### 请求方法

GET `$endpoint/wallets/${wallet_id}/recharges`

#### 请求参数

| 字段名    | 类型   | 描述                                        | 属性     |
| --------- | ------ | ------------------------------------------- | -------- |
| wallet_id | string | 钱包对象 ID，该参数仅需要填写在请求地址内。 | required |

#### 返回参数

返回一个支付对象的分页格式数组。
