# 删除结算对象


#### 请求方法

POST `$endpoint/settle/remove/{id}`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| plug_random_key | string | 随机因子 | required |
| encrypt_pwd   | string  | 加密密文                         | required |

#### 返回参数

返回一个结算对象
