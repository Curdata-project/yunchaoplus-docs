# 查询提现对象

通过提现对象的`id`查询一个已创建的提现对象。

#### 请求方法

GET `$endpoint/wallets/${wallet_id}/withdraws/${id}`

#### 请求参数

| 字段名    | 类型   | 描述                                        | 属性     |
| --------- | ------ | ------------------------------------------- | -------- |
| wallet_id | string | 钱包对象 ID，该参数仅需要填写在请求地址内。 | required |
| id        | string | 提现对象ID，该参数仅需要填写在请求地址内。  | required |
#### 返回参数

返回一个支付对象。
