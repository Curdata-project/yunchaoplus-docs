# 创建结算对象

你可以通过该接口创建一个 `settle` 对象。

#### 请求方法

POST `$endpoint/settle`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| wallet_id | string | 钱包对象id，即结算对象相关关联的钱包对象 | required |
| channel   | string  | 渠道，当前仅支持银行卡（bank_card）                         | required |
| extra           | object      | 额外参数，具体渠道不同有所区别，参见额外参数                 | optional      |


#### 返回参数

返回一个结算对象。
