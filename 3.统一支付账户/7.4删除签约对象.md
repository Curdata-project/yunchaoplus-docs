# 删除签约对象

通过签约对象的`id`删除一个已创建的结算对象。

#### 请求方法

DELETE `$endpoint/wallets/{wallet_id}/agents/${id}`

#### 请求参数

| 字段名    | 类型   | 描述                                        | 属性     |
| --------- | ------ | ------------------------------------------- | -------- |
| wallet_id | string | 钱包对象 ID，该参数仅需要填写在请求地址内。 | required |
| id        | string | 签约对象ID，该参数仅需要填写在请求地址内。  | required |
#### 返回参数

返回一个结算对象。

#### 请求示例
```http request
localhost:8080/wallets/1/settles/1
```

#### 响应示例

```json
{
    "code": 0,
    "message": "success",
    "data": {
        "id": "1",
        "type": null,
        "created": null,
        "channel": null,
        "recipient": null,
        "wallet_id": "1",
        "extra": null
    }
}
```
