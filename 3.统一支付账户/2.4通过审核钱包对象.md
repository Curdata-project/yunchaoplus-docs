# 锁定解锁钱包对象

你可以通过该接口将一个钱包通过审核

#### 请求方法

PUT `$endpoint/wallets/${id}/review`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| id         | string      | 钱包id，该参数仅需要填写在请求地址内， | required      |

#### 返回参数

返回一个钱包对象。
