# 创建钱包对象

你可以通过该接口创建一个 `wallet` 对象。

#### 请求方法

POST `$endpoint/wallet`

#### 请求参数

| 字段名         | 类型        | 描述                                                         | 属性          |
| -------------- | ----------- | ------------------------------------------------------------ | ------------- |
| block | boolean | 是否锁定，默认为false | optional |
| channel | string | 渠道，对应钱包账户服务商的渠道。 | required |
| extra           | object      | B:unipay_merchant ; C:unipay_consumer。对应钱包账户服务商的渠道。              | optional      |

#### 返回参数

返回一个钱包对象。
