# 查询转账对象列表

返回之前创建过提现对象的一个列表。具体的查询参数遵循前述中描述的分页参数。

#### 请求方法

GET `$endpoint/list/transfer`

#### 请求参数

| 字段名 | 类型   | 描述                                        | 属性     |
| ------ | ------ | ------------------------------------------- | -------- |
| page     | int | 页数 | required |
| count     | int | 条数 | required |
| begin_time     | int | 开始时间 | optional |
| end_time     | int | 结束时间 | optional |


#### 返回参数

返回一个转账对象的分页格式数组。

#### 请求示例

/list/transfer?page=1&count=3
