# 查询支付对象

通过支付对象的`id`查询一个已创建的支付对象。

#### 请求方法

GET `$endpoint/charges/{id}`

#### 请求参数

| 字段名 | 类型   | 描述                                        | 属性     |
| ------ | ------ | ------------------------------------------- | -------- |
| id     | string | 支付对象 ID，该参数仅需要填写在请求地址内。 | required |
#### 返回参数

返回一个支付对象。

#### 请求示例
```
http://127.0.0.1:9010/charges/2df5df36-1dd2-11b2-802a-7061796d6e74
```
