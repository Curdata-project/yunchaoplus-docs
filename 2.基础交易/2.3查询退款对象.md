# 查询退款对象

通过退款对象的`id`查询一个已创建的支付对象。

#### 请求方法

GET `$endpoint/charges/{charge_id}/refunds/{refund_id}`

#### 请求参数

| 字段名    | 类型   | 描述                                        | 属性     |
| --------- | ------ | ------------------------------------------- | -------- |
| charge_id | string | 支付对象 ID，该参数仅需要填写在请求地址内。 | required |
| refund_id | string | 退款对象 ID，该参数仅需要填写在请求地址内。 | required |
#### 返回参数

返回一个退款对象。
