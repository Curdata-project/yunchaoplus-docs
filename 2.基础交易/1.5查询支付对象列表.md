# 查询支付对象列表

返回之前创建过支付对象的一个列表。具体的查询参数遵循前述中描述的分页参数。

#### 请求方法

GET `$endpoint/wallet/{wallet_id}/list`

#### 请求参数

| 字段名 | 类型   | 描述                                        | 属性     |
| ------ | ------ | ------------------------------------------- | -------- |
| page     | int | 页数 | required |
| count     | int | 条数 | required |
| begin_time     | int | 开始时间 | optional |
| end_time     | int | 结束时间 | optional |
| status     | string | 状态 | optional |


#### 返回参数

返回一个支付对象的分页格式数组。
